Estrutura básica:

A estrutura básica do HTML é composta pelas tags HTML onde estará composta toda as demais tags para a criação de uma página de texto, seu cabeçalho composto pela tag HEAD, o que o usuário não vê e por fim a tag BODY que é o que será apresentado visualmente ao usuário.

Tag Strong:

Tag utilizada para deixar um texto ou uma palavra em negrito.

Tag I:

Tag utilizada para deixar um texto ou uma palavra em itálico.

Tag H1 a H6:

Tags utilizadas para titulos e subtitulos, derivando o grau da hierarquia da tag H1 até H6.

Lista ordenada:

Composta pela tag mestre OL e suas tags filhas LI.

Exemplo de lista ordenada:

  1. Item 1
  2. Item 2
  3. Item 3

Lista não ordenada:

Composta pela tag mestre UL e suas tags filhas LI.

Exemplo de lista não ordenada:

  • Item 1
  • Item 2
  • Item 3
